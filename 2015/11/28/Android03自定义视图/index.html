Android笔记 03自定义视图

1. 做一个圆形的红色按钮

中间有一个白色的数字
数字起始为20
每点击一次减少1 

  @Override
protected void onDraw(Canvas canvas) {
super.onDraw(canvas);
//做一个红红色按钮
//设置画布为红色
mPaint.setColor(Color.RED);
canvas.drawCircle(getWidth() / 2, getHeight() / 2, getWidth() / 2, mPaint);
//中间有一个白色的数字
mPaint.setColor(Color.WHITE);
mPaint.setTextSize(200);
/**
 * mRect是数字四周的边距
 */

String number = mNunber+"";
mPaint.getTextBounds(number,0,number.length(),mRect);

int textWidth = mRect.width();
int textHeight = mRect.height();

canvas.drawText(number,getWidth()/2-textWidth/2,getHeight()/2+textHeight/2,mPaint);
}

@Override
public void onClick(View v) {

if (mNunber>0){
mNunber--;
}else
mNunber=20;
invalidate();
}

如何自定义视图属性？

  • 在哪里创建属性？

在res/values下新建attrs.xml文件

<?xml version="1.0" encoding="utf-8"?>
<resources>

<declare-styleable name="DrawView">
<attr name="backgroundColor" format="color"/>

</declare-styleable>
</resources>
  • 可以创建哪些属性

format下代表属性的类型,有int,string,color…..

  • 如何使用这些属性

1.代码中引用

typedArray typedArray = context.obtainStyledAttributes(attrs, R.styleable.TestRedButton);//先取出属性集合

mBackgroundColor = typedArray.getColor(R.styleable.TestRedButton_backgroundColor, Color.RED);//再取出某个属性。第二个参数是该属性的默认值

2.布局中设置 

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
xmlns:app="http://schemas.android.com/apk/res-auto"
android:orientation="vertical"
android:layout_width="match_parent"
android:layout_height="match_parent">

<com.geekband.Test01.TestRedButton
android:layout_width="300dp"
android:layout_height="300dp"
app:backgroundColor="@color/splashBackgroundColor"
app:textSize="18dp"
/>

</LinearLayout>

在布局文件中加入 xmlns:app=”http://schemas.android.com/apk/res-auto“ 的命名空间，然后就可以用自定义命名空间里的属性值了。
