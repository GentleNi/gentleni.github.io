Android08_Widget

Menu
  1. What is Widget
  2. 如何创建

What is Widget

窗口小部件，桌面上的n*n的小控件，常见的像音乐播放器控件，360加速等

如何创建

一个class三个xml

  1. 新建 TestWidget 继承自 AppWidgetProvider,可见 Widget是继承自BroadcastReceiver的。然后复写其onReceive方法和onUpdate方法，注意别忘了onUpdate方法，否则加载时候会提示加载小部件出现问题，加载不出来。

  2. Widget的配置xml，setting_widget.xml

    <appwidget-provider xmlns:android="http://schemas.android.com/apk/res/android"

    android:layout_width=”match_parent”
    android:layout_height=”match_parent”
    android:initialLayout=”@layout/layout_widget”
    android:minHeight=”140dp”
    android:minWidth=”140dp”
    android:previewImage=”@drawable/gege”
    android:updatePeriodMillis=”20000”
    android:widgetCategory=”home_screen”
    >

  3. Widget的布局文件 layout_widget.xml

    <LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
  android:orientation="vertical"
  android:layout_width="match_parent"
  android:layout_height="match_parent">

<TextView
android:layout_width="wrap_content"
android:layout_height="wrap_content"
android:textAppearance="?android:attr/textAppearanceLarge"
android:text="Large Text"
android:id="@+id/widget_text_view"
android:layout_gravity="center_horizontal"/>

<Button
android:layout_width="wrap_content"
android:layout_height="wrap_content"
android:text="New Button"
android:id="@+id/widget_button"
android:layout_gravity="center_horizontal"/>
</LinearLayout>

  4. 在AndroidManifest.xml中注册

    <receiver android:name=".TestWidget">
<intent-filter>
<action android:name="android.appwidget.action.APPWIDGET_UPDATE"/>
</intent-filter>
<meta-data android:name="android.appwidget.provider"
android:resource="@layout/setting_widget"
>
</meta-data>
</receiver>

Android06-BroadcastReceiver

Menu
  1. 注册

注册

  • 静态注册
  • 动态注册

1.SendBroadcastActivity.java

public class SendBroadcastActivity extends Activity {

public static final String COM_GENTLENI_TEST_BROADCAST = "com.gentleni.test.broadcast";
private TestBroadcastReceiver mTestBroadcastReceiver = new TestBroadcastReceiver();
private Button mButtonSendBroadcast;
@Override
protected void onCreate(Bundle savedInstanceState) {

    setContentView(R.layout.activity_sendbroadcast);
    mButtonSendBroadcast=(Button)findViewById(R.id.btn_send_broadcast);

    mButtonSendBroadcast.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View v) {

            // 发送广播
            Intent intent = new Intent();
            intent.setAction(COM_GENTLENI_TEST_BROADCAST);
            intent.putExtra("toast", "this is my toast of broadcast");

            sendBroadcast(intent);
        }
    });
    super.onCreate(savedInstanceState);
}

@Override
protected void onStart() {
    super.onStart();
//        动态注册BroadcastReceiver
    IntentFilter intentFilter = new IntentFilter();
    intentFilter.addAction(COM_GENTLENI_TEST_BROADCAST);
    registerReceiver(mTestBroadcastReceiver, intentFilter);
}
}

2.BroadcastReceiver.java

public class TestBroadcastReceiver extends BroadcastReceiver{
    @Override
    public void onReceive(Context context, Intent intent) {

        if (intent != null) {
            if (TextUtils.equals(intent.getAction(), SendBroadcastActivity.COM_GENTLENI_TEST_BROADCAST)) {
                String toastString = intent.getStringExtra("toast");
                Toast.makeText(context, toastString, Toast.LENGTH_LONG).show();
            }
        }
    }
}

3.AndroidManifest.xml

<receiver android:name=".TestBroadcastReceiver">
           <intent-filter>
               <action android:name="com.gentleni.test.broadcast"></action>
           </intent-filter>
       </receiver>
